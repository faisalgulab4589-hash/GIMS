ISSUE:

All tabs in the Student Management System dashboard were not opening.
Users could see tabs but clicking on them had no effect.

ROOT CAUSE:
-----------
In templates/dashboard.html, a 38-line JavaScript code block (lines 101-138)
that handles tab click events was incorrectly placed inside a CSS comment block.

The code looked like this:

    /*
    JavaScript code for tab handling... (incorrectly placed here)
    */
    /* More CSS */

This prevented the tab initialization code from executing.
